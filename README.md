This is the open-source repository for **Embracing Change** by Kai Brockelt. This book is a scientific framework designed to help you reclaim your nutritional autonomy and cut through the massive jungle of misinformation generated by the industrial food complex.

By hosting this project openly on GitHub under a Copyleft license, this book belongs to the public. It is a tool for a health rebellion—free to read, free to fork, and free to improve.

---

## Reclaiming Autonomy

Modern nutrition advice is often broken, corporate-driven, or over-complicated. This book doesn't give you rigid, dogmatic meal plans. Instead, it arms you with the underlying biochemistry and practical tools — so you can become the pilot of your own health.

By moving away from standard copyright laws, this project mirrors its own message: **Information, just like our food supply, should not be locked behind paywalls.**

## How you can Contribute

Because this book is open-source, you are invited to improve it! Whether you are a nutritionist, a developer, a designer, or an avid reader, your help is welcome.

### What you can contribute:

* **Fix Typos & Grammar:** Found a bug in the text? Open a Pull Request.
* **Localization / Translation:** Want to translate Embracing Change into other languages? All files are available in an open format, ready for translations.
* **Community Photos:** I am actively looking to replace stock photos with authentic, community-driven photography. 
* **Extension & Science updates:** Knowledge evolves, so the book shall adapt to new discoveries.

### Tools used to work on the book

All Graphics are available under the [affinity](https://www.affinity.studio/) designer format . afdesign (now “affinity studio”). While not open source, this software can be used for free.

To edit the epub files, i recommend using [sigil](https://sigil-ebook.com/).
